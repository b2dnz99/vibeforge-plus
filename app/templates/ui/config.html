📚 Agent Help

What are agents?
AI assistants (Claude, Codex, etc.) that work on your project via API. Each agent has a unique token scoped to one project.
Token security
Tokens are shown once at creation. Store securely. Use the Download agent-token.txt button for secure file handoff.
Onboarding an agent
1. Create agent here or in Admin Panel
2. Download agent-token.txt file
3. Copy onboarding prompt from Admin
4. Paste prompt into agent session
5. Agent reads token, configures, deletes file
Cycle vs Revoke
Cycle: new token, same identity. Use when token may be compromised.
Revoke: agent deactivated. Must cycle after restore.

Why this tab exists.
The same cert information is visible via any browser's padlock icon, but reaching it takes several clicks and most people never do. Surfacing it here makes the trust-bootstrap step one click. When the install uses a public cert (Let's Encrypt, or a commercial CA) no install is needed — this tab becomes informational only.
